## Overview

This repository provides the implementation artifacts and execution instructions for three federated digital twin exemplars.

This repository is associated with the Engineering of Digital Twins conference (EDTconf) paper titled *"Characterizing Federated Digital Twins through Heterogeneity, Autonomy, and Distribution: A Family of Exemplars"*.

## Repository Structure

```text
.
├── Exemplar_1_Centralized_Federation
├── Exemplar_2_Hierarchical_Federation
└── Exemplar_3_Peer_to_Peer_Federation
```

## Exemplars

Each folder contains its own `README.md` file with specific instructions on how to build, run, and test the corresponding exemplar.

* [Exemplar 1: Centralized Federation](./Exemplar_1_Centralized_Federation/README.md)
* [Exemplar 2: Hierarchical Federation](./Exemplar_2_Hierarchical_Federation/README.md)
* [Exemplar 3: Peer-to-Peer Federation](./Exemplar_3_Peer_to_Peer_Federation/README.md)
